# TechBazaar - E-Commerce Frontend Reference Project

Egy modern, letisztult technológiai webáruház frontend prototípusa. A projekt célja a reszponzív dizájn, a tiszta kódstruktúra és a modern frontend technológiák bemutatása állásinterjúkhoz.

## 📋 Projekt Terv & Útmutató

### 1. Fázis: Tervezés és Előkészületek (Kész)
*   **Grafikai váz (UI/UX):** Wireframe és UI dizájn elkészítése (Figma).
*   **Projekt inicializálás:** Git verziókövetés beállítása és GitHub Pages deployment pipeline kiépítése.

### 2. Fázis: Alap Architektúra (Aktuális)
*   **Szemantikus HTML5:** Keresőbarát (SEO) és akadálymentes (Accessibility) alapok lefektetése.
*   **Stíluskezelés:** CSS Grid és Flexbox architektúra kialakítása a reszponzivitásért.

### 3. Fázis: Kulcsfontosságú Funkciók (Frontend)
*   **Főoldal (Hero szekció):** Kiemelt ajánlatok, kategóriaválasztó (telefonok, tabletek, kiegészítők).
*   **Kosár Kezelés:** Lokális kosár funkció (LocalStorage használatával), ahol a termékek hozzáadhatók és törölhetők.
*   **Termékoldal:** Részletes specifikációs táblázatok, képgaléria váltó.

### 4. Fázis: Optimalizálás és Tesztelés
*   **Teljesítmény:** Képek tömörítése (WebP formátum).
*   **Cross-Browser Tesztelés:** Chrome, Firefox, Safari és Edge kompatibilitás biztosítása.

---

## 🛠️ Alkalmazott Technológiák
*   **HTML5** – Szemantikus struktúra
*   **CSS3 / SASS** – Modern elrendezés (Flexbox, Grid) és reszponzív töréspontok
*   **JavaScript (ES6+)** – Dinamikus termékkezelés, kosár logika és DOM manipuláció
*   **GitHub Pages** – Statikus hoszting és CI/CD automatizáció
